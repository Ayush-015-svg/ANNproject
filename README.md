# ANN Mini Project

This repository contains implementations of basic Artificial Neural Network (ANN) models as part of a mini-project.

## 📌 Projects Included

### 1. Perceptron Model
- Implemented using Python from scratch
- Tested on logic gates (AND gate)
- Demonstrates linear classification

### 2. Competitive Learning Network
- Unsupervised learning model
- Uses winner-takes-all strategy
- Clusters noisy 2D data and self-organizes patterns

### 3. Feedforward Neural Network
- Implemented using TensorFlow/Keras
- Uses Iris dataset for classification
- Demonstrates multi-class classification

---

## ⚙️ Technologies Used
- Python
- NumPy
- Matplotlib
- Scikit-learn
- TensorFlow / Keras

---

## 📊 Results
- Perceptron successfully classifies linearly separable data
- Competitive learning forms clusters from input patterns
- Feedforward network achieves good accuracy on Iris dataset

---

## 🎯 Conclusion
This project demonstrates different types of neural networks:
- Supervised learning (Perceptron, Feedforward)
- Unsupervised learning (Competitive Learning)
